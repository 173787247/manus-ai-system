## 🚀 立即启动（3种方式）

### 方式1: 演示模式（推荐）⭐

```bash
cd manus-ai-system
python run_demo.py
```

选择 `1` 启动Web界面

### 方式2: 直接启动Web界面

```bash
cd manus-ai-system
python main.py
```

### 方式3: Windows一键启动

```bash
cd manus-ai-system
start.bat
```
