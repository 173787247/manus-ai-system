# 🎯 使用说明 - 作为顾客使用系统

## ✅ 系统已准备就绪！

所有功能已测试通过，系统可以正常运行。

---

## 🚀 启动系统（3步）

### 步骤1: 打开命令行

在项目目录打开PowerShell或CMD

### 步骤2: 运行启动命令

```bash
cd manus-ai-system
python run_demo.py
```

### 步骤3: 选择模式

输入 `1` 选择Web界面（推荐）

---

## 📱 访问系统

启动后，系统会显示访问地址，例如：

```
============================================================
系统已启动！
============================================================
访问地址: http://localhost:7861
============================================================
```

**重要**: 如果7860被占用，端口可能是7861、7862等，请使用启动时显示的实际端口。

---

## 🎮 使用系统

### 1. 打开浏览器

访问启动时显示的地址（如 http://localhost:7861）

### 2. 输入任务

在"任务指令"文本框中输入任务，例如：

```
打开记事本，输入"Hello World"
```

### 3. 执行任务

点击 **"执行任务"** 按钮

### 4. 查看结果

在"执行结果"区域查看：
- ✅ 任务状态
- 📊 执行步骤数
- ⏱️ 执行时间
- 📝 详细结果

## ⚠️ 常见问题

### Q1: 端口被占用

**现象**: 启动时显示端口被占用

**解决**: 系统已自动修复，会自动使用其他端口（7861、7862等）

**查看**: 启动时显示的访问地址

### Q2: 浏览器无法访问

**解决**:
1. 检查启动时显示的访问地址
2. 确保地址正确（可能是7861而不是7860）
3. 尝试 http://127.0.0.1:7861

### Q3: 任务执行失败

**解决**:
- 使用更清晰的任务描述
- 检查系统状态（右侧显示）
- 查看错误信息
