# 🚀 快速启动指南 - 作为用户使用

## 方法1: 一键启动（最简单）

### Windows用户

```bash
# 双击运行
start.bat

# 或命令行运行
start.bat
```

### Linux/Mac用户

```bash
chmod +x start.sh
./start.sh
```

---

## 方法2: 演示模式（推荐首次使用）

```bash
python run_demo.py
```

然后选择：
- **1** - Web界面（图形界面，推荐）
- **2** - 命令行界面（文本交互）

---

## 方法3: 直接启动Web界面

```bash
python main.py
```

启动后，浏览器会自动打开或访问：**http://localhost:7860**

---

## 📱 使用步骤

### 步骤1: 启动系统

运行上述任一命令启动系统

### 步骤2: 打开浏览器

访问 **http://localhost:7860**

### 步骤3: 输入任务

在"任务指令"框中输入任务，例如：
```
打开记事本，输入"Hello World"
```

### 步骤4: 执行任务

点击 **"执行任务"** 按钮

### 步骤5: 查看结果

在"执行结果"区域查看任务执行结果

## 🐛 常见问题

### Q: 无法启动

**解决**:
```bash
# 检查Python
python --version

# 安装依赖
pip install -r requirements.txt
```

### Q: 浏览器无法访问

**解决**:
1. 检查是否显示 "Running on http://localhost:7860"
2. 手动访问 http://127.0.0.1:7860
3. 检查防火墙设置

### Q: 任务执行失败

**解决**:
- 使用更清晰的任务描述
- 检查系统状态
- 查看错误信息
