## 🎉 端口问题已解决

系统现在会自动查找可用端口，如果7860被占用，会自动使用7861、7862等。

### 方法1: 演示模式（推荐）

```bash

```

```bash

启动后，系统会显示访问地址，例如：

```

## ⚠️ 如果端口仍被占用

### 方法1: 关闭占用端口的程序

```powershell
# 查看占用7860的程序
netstat -ano | findstr :7860

# 结束进程（替换PID为实际值）
taskkill /PID <PID> /F
```

### 方法2: 使用命令行模式

```bash
python run_demo.py
# 选择 2 (命令行界面，不占用端口)
```

### 方法3: 等待几秒后重试

有时之前的进程还在关闭中，等待几秒后重试。
