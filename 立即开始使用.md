## 🚀 三种启动方式

### 方式1: 演示模式（推荐首次使用）⭐

```bash
cd manus-ai-system
python run_demo.py
```

**然后选择**:
- 输入 `1` → 启动Web界面（图形界面，推荐）
- 输入 `2` → 启动命令行界面（文本交互）

### 方式2: 直接启动Web界面

```bash
cd manus-ai-system
python main.py
```

### 方式3: 使用启动脚本（Windows）

```bash
cd manus-ai-system
start.bat
```

## 🎮 使用流程

### 1️⃣ 输入任务

在"任务指令"文本框中输入您想要执行的任务：

**示例任务**:
```
打开记事本，输入"Hello World"，保存为test.txt
```

```
打开浏览器，访问百度，搜索"AI Agent"
```

```
打开计算器
```

### 2️⃣ 设置参数（可选）

- **最大步骤数**: 1-50（默认10）
- 简单任务可以设置小一点
- 复杂任务可以设置大一点

### 3️⃣ 执行任务

点击 **"执行任务"** 按钮

### 4️⃣ 查看结果

在"执行结果"区域查看：
- ✅ 任务状态（成功/失败）
- 📊 执行步骤数
- ⏱️ 执行时间
- 📝 执行详情

## 🆘 遇到问题？

### 问题1: 无法启动

```bash
# 检查Python
python --version

# 安装依赖
pip install -r requirements.txt
```

### 问题2: 浏览器无法访问

1. 检查控制台是否显示 "Running on http://localhost:7860"
2. 尝试访问 http://127.0.0.1:7860
3. 检查防火墙设置

### 问题3: 任务执行失败

- 使用更清晰的任务描述
- 检查系统状态
- 查看错误信息
